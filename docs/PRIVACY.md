# Privacy Policy Draft — ParkPilot Disneyland Trip Assistant

Last updated: 2026-06-05

ParkPilot Disneyland Trip Assistant is a trip-planning prototype that uses deterministic seed data and local app state to help plan a Disneyland Resort visit.

## Data collected

The current public repository does not include a hosted service and does not collect data from users. The native app prototype may store planning state locally on the user's device when implemented by the app runtime.

## Data not collected

The project does not collect or store Disney account credentials, payment card data, government identifiers, health information, API keys, passwords, MFA codes, or precise location history.

## Third-party services

The prototype includes links that hand users off to official Disney web or app flows for reservations or Lightning Lane actions. It does not automate Disney account actions, scrape Disney services, or submit information to Disney on the user's behalf.

## Retention

No server-side retention applies because no hosted backend is included in this repository. Any local device data should be removable by deleting the app.

## Contact

For support, use the maintainer contact listed on the GitHub repository.
